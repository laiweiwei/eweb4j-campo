eweb4j-campo
============
此项目是使用eweb4j框架所编写的一个小型社区Demo，旨在供大家入门学习eweb4j而用。  
###说明###:  
1.数据库文件存在mysql文件夹内  
2.此项目需用用到一个解析markdown的第三方类库包  
<dependency>  
	<groupId>org.pegdown</groupId>  
	<artifactId>markdown-parser</artifactId>  
	<version>1.2.1</version>  
</dependency>  
需要自己手动导入maven的本地仓库，jar包在jar文件夹下
